task will have: type, description, links, due and time-interval.

when creating, the user will select what type and is possible to create more types when selecting it in a (+) button provided in the select menu.

description will be provided by the user and can contain as so much characters that the usar want;

links will be added by the user and embedded in a name typed by the user. 

due will be selected by the user in a prompt calendar and after in a clock, its possible to set due-date as null as well.

time-interval will be selected by the user in a calendar and a clock as the due element. time-interval is mandatory
